# Segmentación Estratégica de Clientes E-commerce
**Fecha:** 16 de Febrero, 2026
**Autores:** Jefferson Velasquez, Frank Macias, Jorge Murillo
**Asunto:** Identificación de Perfiles de Comportamiento de Compra mediante Clustering (Aprendizaje No Supervisado)

---

## 1. Visión General
El presente análisis utiliza algoritmos de Machine Learning no supervisado (**K-Means, DBSCAN**) validados mediante reducción de dimensionalidad (**PCA** y **t-SNE**) para segmentar la base de clientes de una plataforma de e-commerce con servicios personalizados.
El objetivo es identificar los grupos de clientes de la plataforma con el fin de brindar una mirada al equipo de marketing para que puedan enfocar sus campañas en grupos segmentados, entendiendo el comportamiento de los mismos, en vez de hacer creación genérica de contenido.
